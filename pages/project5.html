Odyssey

Info:

Senior design project apart of CPRE 491. This project creates a centralized trip planning application that allows organizations to advertise trips and appoint trip leaders to manage them. Trip goers can view and sign up for trips, and the trip leader can manage the trip details and attendees.

I was responsible for the backend development and security of the project. The backend was built using Express.js and PostgreSQL hosted on a VM. Nginx was used as a reverse proxy and to handle SSL.
