Embedded Systems Cybot

Info:

This project was apart of CPRE 288 - Introduction to Embedded Systems. We used ultrasonic and infrared sensors, an ARM Cortext M4 based microcontroller, and a Roomba chassis to assemble a robot capable of being remotely controlled inside a predetermined course.

The project was completed in a group of four, and I was responsible for writing the code to control the robot, merging group members' contributions into our main repository, and I wrote a program to perform an automatic calibration sequence for the sensors and motors used by the microcontroller. I also helped with programming the commandline GUI used for controlling the robot and visualizing the sensor data.

This course project taught me how C code converts to assembly, GPIO, interrupt services, and better practices for reading documentation and datasheets. Additionally, I learned how to use GitHub for version control and collaboration in a group project.

Resources: C, assembly, sensor datasheets, microcontroller documentation, GitHub, Cygwin
