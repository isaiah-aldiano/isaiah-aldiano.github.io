Embedded Systems Cybot

Info:

This project used ultrasonic and infrared sensors, an ARM Cortext M4 based microcontroller, and a Roomba chassis to assemble a robot capable of being remotely controlled inside a predetermined course.

The project was completed in a group of 4, and I was responsible for writing the code to control the robot, as well as integrating the sensors and motors with the microcontroller. I also helped with programming the commandline GUI used for controlling the robot and visualizing the sensor data. This course project taught me how C code converts to assembly, GPIO, interrupt services, and better practices for reading documentation and datasheets.
