Agora Marketplace

Info:

This project was apart of Coms 309 - Software Development Practices. The project was to create a marketplace application that allows users to buy and sell items locally instead of shipping. The application was built using Java with Volley on the frontend and SpringBoot on the backend. The application also uses MariaDB for the database.

The project was developed in a team of four. A group member and I were responsible for the frontend development of the application. I was responsible for implementing the user interface, integrating backend APIs using JSON Volley requests, and documenting on 35+ activities.

Resources: Java, Volley, SpringBoot, MariaDB, GitHub, Android Studio, Figma
